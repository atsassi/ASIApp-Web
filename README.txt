ASIApp Render — PWA
-------------------
1. Comprimi questa cartella come ASIApp_Render_Final.zip.
2. Vai su https://dashboard.render.com → New → Static Site.
3. Carica lo ZIP (oppure incolla link pubblico).
4. Dopo il deploy, apri l’URL tipo https://asiapp.onrender.com
5. Login: atsassi@gmail.com / ASI2025! oppure demo@asiapp.ch / DEMO2025
6. Per collegare asiapp.ch (GoDaddy):
   Type: CNAME
   Name: www
   Value: asiapp.onrender.com
   TTL: 600
7. Per attivare le API Alpaca reali, modifica app.js e integra
   le funzioni di alpaca_example.js dove indicato.
